---Инструкция для установки программы---
1. Прежде чем начинать работать с программой, вам необходимо установить интерпретатор python. 
   Если он у вас уже есть, то хорошо, если нет, то вот инструкция.
2. Заходите на официальный сайт https://www.python.org. Там находите кнопку Downloads и скачиваете для Windows или Mac (там сразу вам предлагается вариант).
2. На первой странице установки установите флажок внизу, рядом надписью add Python38 to PATH.
   А также проверьте, что первая галочка тоже стоит. После, нажмите Install Now.
3. После установки у вас будет последняя страница установщика с предложением об обучении и т.д.
   Вам нужно нажать на нижний текст, где написано MAX path limit. После завершения этого этапа можете 
   нажать конпку "Close".
4. Теперь откройте командную строку, нажав на клавишу Windows и набрав (просто) cmd, затем нажмите Enter.
5. В открывшемся окне напишите выражение pip install openpyxl (можете скопировать отсюда).
   Подождите немного, пока компьютер загрузит необходимые компоненты.
6. После того, как вы увидете желтую надпись, можете закрывать консоль.
7. Проверьте, правильно ли все работает, попробовав открыть файл с программой (у него 
   расширение .py) через "Открыть с помощью" -> Python. У вас должно открыться одно черное окно,
   после которого вылезет разрешение на внос изменений. Вы нажимаете Да и во втором окне жмете Enter.
   Теперь должно открыться окошко с кнопками. Попробуйте ввести какие-нибудь буквы в поля (английские) и нажать "Создать".
   Должно появится сообщение: "Готово!". Если все ок, то переходите дальше, вы молодец :)

---Для создания таблиц---
1. В поисковой строке hh.ru введите запрос по кандидату.
2. В открывшемся окне с результатами поиска нажмите Ctrl + U. У вас в соседней
   вкладке откроется огромный текст - это код страницы.
3. Зайдите на эту вкладку и нажмите Ctrl + A, чтобы выделить весь текст этой страницы.
4. Далее, скопируйте выделенный текст сочетанием клавиш Ctrl + C, либо через нажатие
   правой кнопкой мыши.
5. В папке с приложением откройте файл htmltext.txt и вставте скопированный текст туда (там есть буква A, её можно удалить или не обращать внимание).
6. Если у вас с полученном запросе несколько страниц с резюме, то повторите шаги 2-5
   для каждой страницы поиска, копируя содержимое в файл htmltext.txt.
7. После того, как вы закончили предыдущие шаги, сохраните документ и закройте его.
8. Теперь вам нужно открыть файл с расширением '.py' с помощью программы python (щелкните
   правой кнопкой мыши по файлу -> "Открыть с помощью" (установите флажок чтобы открывать файл с расширением .py по умолчанию). Должно произойти тоже самое, что и в 
   последнем шаге с установкой.
9. В соответствующих полях введите название файла и дату создания. Нажмите на кнопку
   "Создать".
10. Готово! :)

P.S. Для разных запросов нужно будет очищать файл htmltext.txt. Поэтому, если вы сдедали
таблицу под одну вакансию, то можете, не закрывая приложение, нажать кнопку "Очистить
файл", чтобы сделать таблицу для других резюме.
